# Personal Site
Private site for my portfolio. 
React / Next.js

## Local Running

### Requirements
You need npm installed

### .env File

.env file 
```bash
touch .env
```

Sample .env
```bash
model_host="0.0.0.0"
model_port=8000
email="email@provider.com"
phone="1234567890"
```

## Run the Next.js App

### .env.local file
```bash
cd nextjs

touch .env.local
```

```bash

npm run dev

```
